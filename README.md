# nestjs-microservice-server

## Description

This is an example repo for:

- [Nest](https://github.com/nestjs/nest) server side [microservices](https://docs.nestjs.com/microservices/basics)
- Unit test and e2e test
- GitHub actions and Codecov setup (Badge at README.md)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
