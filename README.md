## Description

**Example repo for**

- [Nest](https://github.com/nestjs/nest) framework TypeScript Microservices
- GitHub actions and Codecov setup

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
